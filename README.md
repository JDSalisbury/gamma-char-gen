## Gamma World Character Generator

```
User Logs in.
	- Can view Characters or create new.

Create Char
	user presented with three choices.
	user pics choice and that creates the character
		- then asked to roll stats?

View all Character sheets
    Selecting Char goes into Character sheet details
	- this view shows stats.
	- Note like area for inventory
	- can roll weapon attacks here
	- can lvl character and get new choices when at certain lvls. Other choices become available after going
			over.
```
